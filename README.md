# Console Battle Arena (Python OOP)

**"A strategic turn-based console game developed with Python and OOP principles."**

This project is a battle simulation game that runs in your terminal. It features 10 unique character classes, a dynamic battlefield, and strategic combat mechanics. It is designed to demonstrate Object-Oriented Programming concepts like Inheritance and Polymorphism.
